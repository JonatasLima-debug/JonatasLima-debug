## Projetos em destaque

### Previsão de Diabetes com Machine Learning
Pipeline completo de ML para classificação de risco de diabetes utilizando o dataset Pima Indians Diabetes. Inclui pré-processamento, comparação de 6 algoritmos, ajuste fino com GridSearchCV e dashboard interativo no Power BI integrado com o modelo via script Python.

**Tecnologias:** Python · scikit-learn · SMOTE · Random Forest · Power BI
